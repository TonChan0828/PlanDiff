# 指示資料置き場

ユーザーが開発を指示するための資料(メモ、画面ラフ、参考記事、仕様の要望など)を置く場所。
形式は自由(md / 画像 / PDF など)。

## 運用

- ファイル名は `YYYY-MM-DD_<内容>.<拡張子>` を推奨(例: `2026-07-05_タイマーUIのラフ.png`)
- Claude は **タスク着手時に必ずこのディレクトリを確認**し、関連する資料があれば仕様書に反映する
- 資料を仕様書に取り込んだら、仕様書の「関連」欄に資料パスを記載する(資料自体は消さない)
- 資料の内容と要件定義書・CLAUDE.md が矛盾する場合、Claude は勝手に解釈せずユーザーに確認する
